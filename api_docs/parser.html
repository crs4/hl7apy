Parser¶

hl7apy.parser.parse_message(message, validation_level=None, find_groups=True, reference=None)¶

Parse the given ER7-encoded message and return an instance of hl7apy.core.Message.

Parameters:
  • message (basestring) – the ER7-encoded message to be parsed
  • validation_level – the validation level. Possible values are those defined in hl7apy.consts.VALIDATION_LEVEL class or None to use the default validation level (see hl7apy.set_default_validation_level())
  • find_groups (bool) – if True, automatically assign the segments found to the appropriate hl7apy.core.Group instances. If False, the segments found are assigned as children of the hl7apy.core.Message instance
Returns:

an instance of hl7apy.core.Message

 
>>> message = "MSH|^~\&|GHH_ADT||||20080115153000||OML^O33^OML_O33|0123456789|P|2.5||||AL\rPID|1||566-554-3423^^^GHH^MR||EVERYMAN^ADAM^A|||M|||2222 HOME STREET^^ANN ARBOR^MI^^USA||555-555-2004|||M\r"
>>> m = parse_message(message)
>>> print m
<Message OML_O33>
>>> print m.msh.sending_application.to_er7()
GHH_ADT
>>> print m.children
[<Segment MSH>, <Group OML_O33_PATIENT>]

Returns:

a list of hl7apy.core.Segment instances

 
>>> segments = "EVN||20080115153000||||20080114003000\rPID|1||566-554-3423^^^GHH^MR||EVERYMAN^ADAM^A|||M|||2222 HOME STREET^^ANN ARBOR^MI^^USA||555-555-2004|||M\r"

hl7apy.parser.parse_fields(text, name_prefix=None, version=None, encoding_chars=None, validation_level=None, force_varies=False)¶

Parse the given ER7-encoded fields and return a list of hl7apy.core.Field.

Parameters:
  • text (basestring) – the ER7-encoded string containing the fields to be parsed
  • name_prefix (basestring) – the field prefix (e.g. MSH)
  • version (basestring) – the HL7 version (e.g. “2.5”), or None to use the default (see hl7apy.set_default_version())
  • encoding_chars (dict) – a dictionary containing the encoding chars or None to use the default (see hl7apy.set_default_encoding_chars())
  • validation_level – the validation level. Possible values are those defined in hl7apy.consts.VALIDATION_LEVEL class or None to use the default validation level (see hl7apy.set_default_validation_level())
Returns:

a list of hl7apy.core.Field instances

 
>>> fields = "1|NUCLEAR^NELDA^W|SPO|2222 HOME STREET^^ANN ARBOR^MI^^USA"
>>> nk1_fields = parse_fields(fields, name_prefix="NK1")
>>> print nk1_fields
[<Field NK1_1 (SET_ID_NK1) of type SI>, <Field NK1_2 (NAME) of type XPN>, <Field NK1_3 (RELATIONSHIP) of type CE>, <Field NK1_4 (ADDRESS) of type XAD>]
>>> s = Segment("NK1")
>>> s.children = nk1_fields
>>> print s.to_er7()
NK1|1|NUCLEAR^NELDA^W|SPO|2222 HOME STREET^^ANN ARBOR^MI^^USA
>>> unknown_fields = parse_fields(fields)
>>> s.children = unknown_fields
>>> print s.to_er7()
NK1|1|NUCLEAR^NELDA^W|SPO|2222 HOME STREET^^ANN ARBOR^MI^^USA
