HL7apy’s documentation¶

HL7apy is a Python package to parse, create and handle HL7 v2 messages.

Currently supported versions are: 2.2, 2.3, 2.3.1, 2.4, 2.5, 2.5.1, 2.6.

HL7apy is developed and maintained by Healthcare Flows researchers at CRS4.

Installation¶

HL7apy is platform independent and supports Python 2.7

To install it get the latest release from here, extract it and launch the following command from the extracted directory

python setup.py install
